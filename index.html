ember-semantic-ui

A variety of semantic-ui components for ember app

Demo Download latest v0.0.9

Integrateed with semantic-ui

Based on semantic-ui, support many of semantic-ui components

Be easily used in ember app

Especially custome for ember component, very easily be used in ember app

Getting started

In your ember-cli project, install this addon from npm

or install the latest version from github

Use this addon in your ember application

Modify app/index.html and import semanti-ui js file and css file

Start ember app

Then you can use this addon components in your ember app
